# Homebrew Formulae

A [Homebrew tap] for various projects.

[Homebrew tap]: https://github.com/Homebrew/brew/blob/master/docs/brew-tap.md

## Usage

Tap this repository:

    brew tap neezer/formulae

Install the packages it contains like any other Homebrew package:

    brew install tinge
    brew install peon
